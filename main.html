U.S. State Innovation Rate (2013)

Hover on a state to see the its innovation rate.
Reveal the entire choropleth map by exploring all the states.

Description

This is an interactive map that shows the result of patenting outcomes for children born in 1980-1984 by state in which they grew up and their gender distribution. The choropleth is filled by scaling the innovation rate (number of inventors / number of children born) in each state.
Data source: Opportunity Insights
